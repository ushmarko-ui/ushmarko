Projet FUNKYTOWN

Interconnexion sécurisée multi-sites et déploiement IPv6

I. Contexte du Projet

Simulation de l'interconnexion de quatre sites géographiques (Aya, Jules, Erwan et eXia) au sein d'une infrastructure d'entreprise distribuée via un cœur de réseau FAI.

Routage : IPv6 Unicast-Routing
Services : DNS, FTP, HTTP
Sécurité : Contrôle d'accès RBAC
Outil : Cisco Packet Tracer

II. Focus : Architecture du Site Engie

Conception d'un LAN haute densité pour le site Engie comprenant :

  • Structure : Topologie en étoile avec cascade de switchs.
  • Mobilité : Bornes Wi-Fi (Access Points) pour terminaux mobiles.
  • Services : Serveur local dédié (ENGIE_SERVER).
  • Segmentation : Isolation du trafic filaire et nomade.

III. Configuration et Tests de Connectivité

Mise en place du routage moderne pour assurer la communication entre les sites distants et les ressources partagées.

Aperçu Configuration (CLI Cisco)

! Activation du routage IPv6
ipv6 unicast-routing

! Configuration de l'interface vers le LAN
interface GigabitEthernet0/0
 ipv6 address 2001:DB8:ACAD:1::1/64
 ipv6 enable
 no shutdown

Résultat : Validation complète par tests de "Pings" inter-sites réussis, garantissant l'accès aux serveurs DNS et FTP globaux.
