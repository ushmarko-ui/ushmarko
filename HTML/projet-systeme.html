Worldwide Weather Watcher

Station météo embarquée pour navires AIVM (C++ Arduino)

Note technique : J'ai intégralement programmé l'interaction de chaque composant hardware. De la gestion du bus I2C pour les capteurs à la communication série pour le GPS, chaque ligne de code a été conçue pour transformer des modules électroniques bruts en une maquette autonome capable de diagnostiquer ses propres erreurs.

I. Analyse Fonctionnelle & Besoins

Analyse "Bête à corne"

Le projet répond au besoin de fournir des données météo fiables en temps réel pour prévenir les catastrophes naturelles.

Diagramme des Cas d'Utilisation

L'utilisateur peut basculer entre deux modes principaux :

  • Collecter les données : Mode automatique (Standard/Eco).
  • Maintenance : Accès aux données et changement de carte SD.

II. Architecture Hardware & Capteurs

J'ai codé l'interface de chaque module Arduino pour assurer une collecte précise :

Capteur BME280 : Pression, Température, Humidité
Module GPS : Localisation temps réel
Horloge RTC : Horodatage via I2C
Lecteur SD : Logs de données
Chainable LED : Codes couleurs d'état
Boutons : Gestion des pressions longues (5s)

III. Logique de Programmation (C++ Arduino)

Auto-Diagnostic (RTC & GPS)

// Diagnostic de la RTC via le bus I2C
Wire.beginTransmission(0x68);
byte erreurRTC = Wire.endTransmission();

if (erreurRTC != 0) {
    rtcError = true; // Déclenche le clignotement d'alerte Rouge/Bleu
}

Changement de Mode

if (pressDure >= 5000) { 
    modeActuel = 3; // Mode Maintenance activé par pression de 5s
    updateLED(); 
}
