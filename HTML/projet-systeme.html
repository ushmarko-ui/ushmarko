Worldwide Weather Watcher

Station météo embarquée pour navires de surveillance (AIVM)

I. Architecture Hardware

Le système repose sur une carte Arduino UNO et plusieurs modules de précision :

BME280 : Pression, Température, Humidité
GPS : Coordonnées en temps réel
RTC : Horodatage précis (Heure/Date)
SD Card : Stockage local des logs
LED RGB : Indicateur d'état (Standard / Erreur)
Capteur de Lumière : Luminosité ambiante

II. Analyse et Cas d'Utilisation

Bête à corne & Use Cases

L'objectif est de fournir des données fiables pour éviter les catastrophes naturelles.

  • Mode Standard : Collecte et stockage automatique toutes les 10 minutes.
  • Mode Maintenance : Accès aux données et changement de carte SD.
  • Interface LED : Feedback visuel de l'état du système.
